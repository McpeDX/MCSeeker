# 🛠️ `mcseeker` – Advanced Minecraft Server Scanner ![GitHub license](https://img.shields.io/badge/license-GPL--3.0-blue?style=for-the-badge) ![Node.js Version](https://img.shields.io/badge/node-%3E%3D16.0.0-green?style=for-the-badge) ![NPM Downloads](https://img.shields.io/npm/dt/mcseeker?style=for-the-badge) > 🔍 A powerful, modern Minecraft server scanner with GeoIP lookup, bot joining, CSV output, and CLI support. --- ## 📌 Features - ✅ Scan Minecraft servers across IP ranges or single IPs - ✅ Filter by version (`1.20.*`, `1.19.4`, etc.) - ✅ Filter by minimum/maximum players - ✅ Auto-download GeoIP database via MaxMind (with license key) - ✅ Show geolocation (country code + optional coordinates) - ✅ Join servers as a bot and read chat messages - ✅ Output results to CSV, TXT, or log to console - ✅ Support for custom ports (e.g., `25565-25567`) - ✅ Global CLI install: `mcseeker --ip 192.168.1.0/24` - ✅ Clean, modular, well-documented codebase --- ## 🚀 Quick Start ### Install via npm: ```bash npm install -g mcseeker ``` > Or use locally: > > ```bash > npm install > ``` ### Run a scan: ```bash mcseeker --ip 192.168.1.0/24 --port 25565 --min-players 1 --out results.csv ``` > This will scan all devices in the `192.168.1.0/24` range on port `25565`, filter servers with at least 1 player, and save results to `results.csv`. --- ## 🔧 Usage Examples | Command | Description | |-------|------------| | `mcseeker --ip 192.168.1.0/24` | Scan entire subnet | | `mcseeker --ip 1.1.1.1 --port 25566` | Scan one specific server | | `mcseeker --version "1.20.*"` | Only show 1.20.x servers | | `mcseeker --min-players 5` | Servers with 5+ players | | `mcseeker --geo-ip --maxmind-key YOUR_KEY_HERE` | Add country codes using MaxMind | | `mcseeker --enable-client --client-token ABC123XYZ` | Connect as a bot and read chat | | `mcseeker --format txt --log-desc` | Output raw text with descriptions | | `mcseeker --quiet --out results.csv` | Silent mode (only write to file) | > 💡 You can also run it directly: > > ```bash > node scanner.js --ip 192.168.1.0/24 --out results.csv > ``` --- ## 📁 Output Formats | Format | Example | |------|--------| | `csv` (default) | `192.168.1.10:25565,1.20.1,5/20,"Welcome!",US` | | `txt` | `192.168.1.10:25565 1.20.1 5 of 20` | | `txt-connect-only` | `192.168.1.10:25565` | Use `--format csv` or `--format txt` to change. --- ## 🌍 GeoIP Support `mcseeker` can detect the country of each server using MaxMind’s GeoLite2 database. ### To enable: ```bash mcseeker --geo-ip --maxmind-key YOUR_LICENSE_KEY_HERE ``` > 🔐 Get your free MaxMind license key here: [https://www.maxmind.com/en/accounts](https://www.maxmind.com/en/accounts) > ⚠️ The database is downloaded automatically if missing. --- ## 🤖 Bot Integration Enable automatic bot login to join servers and read chat: ```bash mcseeker --enable-client --client-token YOUR_TOKEN_HERE ``` Or let `mcseeker` fetch a free token from [TheAltening](https://api.thealtening.com/free/generate): ```bash mcseeker --enable-client ``` > 📝 The bot logs messages like `[BOT] Player: Hello!` --- ## 📦 Dependencies | Package | Purpose | |-------|--------| | `evilscan` | Fast port scanning | | `minecraft-status` | Ping servers for status | | `mineflayer` | Connect as a bot | | `maxmind` | Geolocation lookup | | `commandos` | Parse CLI arguments | | `node-mcpe-color-parser` | Handle MCPE color codes | All managed via `npm install`. --- ##
